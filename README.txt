				-Rouler pour mieux conduire-

Résumé de l'application :

Permettre aux jeunes permis de suivre leur compteur kilomètrique pendant leur
3 ans de période probatoire.
But : garantir une experience de conduite d'au moins 6000km.

De quoi est-il consitué :
	un splashscreen d'accueil
	un login ( avec register et login)
	application de suivi gps ( avec vitesse, distance et date ).

On active l'application, puis en se déplaçant, elle enregistre nos données gps
avec la vitesse la distance et la date. 

Codée en JAVA et Kotlin sur android studio.

Nous avons pu nous appuyer sur Dagger-hilt, une architecture Mvvm, un système coroutines etc...

Pour lancer l'application il suffit de SYNC le gradle et build le projet sur un device.
